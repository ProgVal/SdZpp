Bienvenue sur PhonyProxy-SdZ.
PhonyProxy est un site web qui affiche des version allégées de certains autres sites, de façon optimisée pour les mobiles.
PhonyProxy est un logiciel libre, développé avec Django. Le code source, ainsi que la documentation et le bug tracker sont disponibles sur GitHub.

Veuillez noter que PhonyProxy n'est en aucun cas affilié aux sites webs présentés, et ne peut être tenu pour responsable du contenu des sites et/ou des actions des utilisateurs.
